# Event Check-In Dashboard

## Tech Stack

- ReactJS
- Vite
- Material UI
- React Router DOM
- Axios
- JSON Server

## Features

- Login Authentication
- Dashboard
- Customer Management (CRUD)
- Booth Management
- Customer Check-In / Check-Out
- QR Code
- Analytics
- Activity Tracking
- Responsive UI
